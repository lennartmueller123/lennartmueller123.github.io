This project is a Markowitz Portfolio Optimizer built using Python and the Dash framework. It provides an interactive web application that allows users to create optimized investment portfolios based on their selected:
- Investment Time Horizon (e.g., 3 months, 1 year, 5 years).
- Risk Preference (e.g., Max Sharpe Ratio, Min Volatility, or targeting a Specific Risk level).

Key Features:

  1. Portfolio Optimization:
    Uses financial data from Yahoo Finance to calculate expected returns and covariance matrices, employing the PyPortfolioOpt library for optimization.
  2. Performance Comparison:
    Visualizes the cumulative returns of the optimized portfolio versus the Dow Jones Index.
  3. Customizable Risk Levels:
    Allows users to target specific levels of volatility if desired.
  4. Performance Metrics:
    Provides detailed statistics such as Sharpe Ratio, Annual Returns, Volatility, and more, comparing the optimized portfolio to the benchmark index.
The app is styled with a futuristic black-and-green color scheme and includes interactive components like dropdowns, input fields, and graphs, making it easy to explore different portfolio strategies.
