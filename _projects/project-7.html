This capstone project from the Google Advanced Data Analytics Certificate addresses high employee turnover at Salifort Motors by building predictive models and identifying key factors influencing attrition.

Key Features:

  1. Exploratory Data Analysis (EDA):
    Detailed visualizations and summary statistics revealed patterns in satisfaction levels, project load, and work hours across departments.
  2. Feature Engineering:
    Converted categorical variables, removed outliers, and created binary features like overworked for enhanced model performance.
  3. Logistic Regression:
    Built an interpretable statistical model to establish baseline accuracy (~83%) and reveal linear relationships between features and turnover.
  4. Tree-Based Machine Learning:
    Tuned decision tree and random forest classifiers via GridSearchCV. The random forest achieved:
    • Accuracy: 98.1%
    • Precision: 96.4%
    • Recall: 91.9%
    • F1-score: 94.1%
  5. Feature Importance & Insights:
    Top predictors included last_evaluation, number_project, tenure, and overworked. Insights led to actionable HR recommendations.
  6. Business Impact:
    The model helps reduce turnover costs by identifying high-risk employees early and improving retention through targeted HR strategies.
