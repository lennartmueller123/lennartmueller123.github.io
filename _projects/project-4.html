This project is a Markowitz Portfolio Optimizer built using Python and the Dash framework. It provides an interactive web application that allows users to create optimized investment portfolios based on their selected:

  • Investment Time Horizon: 3 Months, 1 Year, 5 Years or 10 Years
  • Risk Preference: Max Sharpe Ratio, Min Volatility, or Specific Risk level

Key Features:

  1. Portfolio Optimization: Uses Yahoo Finance + PyPortfolioOpt
  2. Performance Comparison: Cumulative return vs Dow Jones
  3. Customizable Risk Levels: User-specified volatility
  4. Performance Metrics: Sharpe, Returns, Volatility, etc.
