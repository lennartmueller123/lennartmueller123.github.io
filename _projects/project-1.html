This project is an Ontology Extraction tool built with Python and LangChain, leveraging large language models to convert structured or unstructured data (text, PDF, or CSV) data into knowledge graphs and ontologies.

Key Features:

  1. Flexible Input Handling:
    Automatically parses text, PDF, and CSV files, splitting large documents into manageable chunks for reliable processing.
  2. LLM-Powered Extraction:
    Uses LangChainâ€™s LLMGraphTransformer with models like Google Gemini or a local Ollama instance to identify entities and relationships.
  3. Iterative Ontology Building:
    Processes each chunk through multiple extraction iterations, stopping when no new triplets are found to ensure completeness.
  4. Ontology Enrichment:
    Infers and adds hierarchical SUBCLASS_OF relations automatically, building richer ontological hierarchies.
  5. Data Aggregation & Export:
    Consolidates all extracted triplets into a Pandas DataFrame for deduplication, analytics, or export (e.g., CSV).
  6. Visualization Suite:
    - Static graphs via NetworkX and Matplotlib
    - Interactive web UI (Vis.js) for exploring both the raw knowledge graph and the extracted ontology, complete with table views and debug output.
